# Currency Converter BRL

Projeto desenvolvido com HTML, CSS e JavaScript que realiza a conversão de moedas estrangeiras para reais. A aplicação permite selecionar uma moeda, informar um valor e visualizar o resultado convertido.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- SVG

## ✨ Funcionalidades

- Conversão para reais
- Seleção de moeda
- Entrada de valor personalizada
- Exibição dinâmica do resultado
- Interface moderna
- Efeito visual com ondas em SVG
- Layout responsivo

## 📂 Estrutura do Projeto

```bash
CurrencyConverterBRL/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html`.
4. Digite um valor.
5. Selecione uma moeda.
6. Clique em "Converter em reais".
